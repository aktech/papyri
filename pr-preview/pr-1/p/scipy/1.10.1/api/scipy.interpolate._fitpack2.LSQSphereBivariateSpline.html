Examples

Suppose we have global data on a coarse grid (the input data does not have to be on a grid): This example is valid syntax, but we were not able to check execution
>>> from scipy.interpolate import LSQSphereBivariateSpline
... import numpy as np
... import matplotlib.pyplot as plt
This example is valid syntax, but we were not able to check execution
>>> theta = np.linspace(0, np.pi, num=7)
... phi = np.linspace(0, 2*np.pi, num=9)
... data = np.empty((theta.shape[0], phi.shape[0]))
... data[:,0], data[0,:], data[-1,:] = 0., 0., 0.
... data[1:-1,1], data[1:-1,-1] = 1., 1.
... data[1,1:-1], data[-2,1:-1] = 1., 1.
... data[2:-2,2], data[2:-2,-2] = 2., 2.
... data[2,2:-2], data[-3,2:-2] = 2., 2.
... data[3,3:-2] = 3.
... data = np.roll(data, 4, 1)
We need to set up the interpolator object. Here, we must also specify the coordinates of the knots to use. This example is valid syntax, but we were not able to check execution
>>> lats, lons = np.meshgrid(theta, phi)
... knotst, knotsp = theta.copy(), phi.copy()
... knotst[0] += .0001
... knotst[-1] -= .0001
... knotsp[0] += .0001
... knotsp[-1] -= .0001
... lut = LSQSphereBivariateSpline(lats.ravel(), lons.ravel(),
...                                data.T.ravel(), knotst, knotsp)
As a first test, we'll see what the algorithm returns when run on the input coordinates This example is valid syntax, but we were not able to check execution
>>> data_orig = lut(theta, phi)
Finally we interpolate the data to a finer grid This example is valid syntax, but we were not able to check execution
>>> fine_lats = np.linspace(0., np.pi, 70)
... fine_lons = np.linspace(0., 2*np.pi, 90)
... data_lsq = lut(fine_lats, fine_lons)
This example is valid syntax, but we were not able to check execution
>>> fig = plt.figure()
... ax1 = fig.add_subplot(131)
... ax1.imshow(data, interpolation='nearest')
... ax2 = fig.add_subplot(132)
... ax2.imshow(data_orig, interpolation='nearest')
... ax3 = fig.add_subplot(133)
... ax3.imshow(data_lsq, interpolation='nearest')
... plt.show()
