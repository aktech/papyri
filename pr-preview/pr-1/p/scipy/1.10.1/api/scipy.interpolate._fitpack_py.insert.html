You can insert knots into a B-spline. This example is valid syntax, but we were not able to check execution
>>> from scipy.interpolate import splrep, insert

... x = np.linspace(0, 10, 5)
... y = np.sin(x)
... tck = splrep(x, y)
... tck[0]

>>> tck_inserted = insert(3, tck)
