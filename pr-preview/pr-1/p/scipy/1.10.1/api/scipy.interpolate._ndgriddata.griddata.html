>>> grid_x, grid_y = np.mgrid[0:1:100j, 0:1:200j]
but we only know its values at 1000 data points: This example is valid syntax, but we were not able to check execution
>>> rng = np.random.default_rng()
... points = rng.random((1000, 2))
... values = func(points[:,0], points[:,1])
This can be done with `griddata` -- below we try out all of the interpolation methods: This example is valid syntax, but we were not able to check execution
>>> from scipy.interpolate import griddata
... grid_z0 = griddata(points, values, (grid_x, grid_y), method='nearest')
... grid_z1 = griddata(points, values, (grid_x, grid_y), method='linear')
... grid_z2 = griddata(points, values, (grid_x, grid_y), method='cubic')
One can see that the exact result is reproduced by all of the methods to some degree, but for this smooth function the piecewise cubic interpolant gives the best results: This example is valid syntax, but we were not able to check execution
