We can verify the maximum singular value of `m` by computing the maximum length of `m.dot(u)` over all the unit vectors `u` in the (x,y) plane. We approximate "all" the unit vectors with a large sample. Because of linearity, we only need the unit vectors with angles in [0, pi]. This example is valid syntax, but we were not able to check execution
>>> t = np.linspace(0, np.pi, 2000)
... u = np.array([np.cos(t), np.sin(t)])
... np.linalg.norm(m.dot(u), axis=0).max()
4.2809152422538475
`p` is a projection matrix with rank 1. With exact arithmetic, its singular values would be [1, 0, 0, 0]. This example is valid syntax, but we were not able to check execution
>>> v = np.array([0.1, 0.3, 0.9, 0.3])
... p = np.outer(v, v)
... svdvals(p)
array([  1.00000000e+00,   2.02021698e-17,   1.56692500e-17,
         8.15115104e-34])
The singular values of an orthogonal matrix are all 1. Here, we create a random orthogonal matrix by using the `rvs()` method of `scipy.stats.ortho_group`. This example is valid syntax, but we were not able to check execution
>>> from scipy.stats import ortho_group
... orth = ortho_group.rvs(4)
... svdvals(orth)
array([ 1.,  1.,  1.,  1.])
