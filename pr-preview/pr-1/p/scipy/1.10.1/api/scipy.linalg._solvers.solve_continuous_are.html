solve_continuous_are(a, b, q, r, e=None, s=None, balanced=True)
$$X A + A^H X - X B R^{-1} B^H X + Q = 0$$
$$E^HXA + A^HXE - (E^HXB + S) R^{-1} (B^HXE + S^H) + Q = 0$$
