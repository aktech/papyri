Let us consider the problem of minimizing the Rosenbrock function. This function (and its respective derivatives) is implemented in `rosen` (resp. `rosen_der`, `rosen_hess`) in the `scipy.optimize`. This example is valid syntax, but we were not able to check execution
>>> from scipy.optimize import minimize, rosen, rosen_der
A simple application of the *Nelder-Mead* method is: This example is valid syntax, but we were not able to check execution
>>> x0 = [1.3, 0.7, 0.8, 1.9, 1.2]
... res = minimize(rosen, x0, method='Nelder-Mead', tol=1e-6)

array([ 1.,  1.,  1.,  1.,  1.])

Next, consider a minimization problem with several constraints (namely Example 16.4 from [5]_). The objective function is: This example is valid syntax, but we were not able to check execution
>>> fun = lambda x: (x[0] - 1)**2 + (x[1] - 2.5)**2
There are three constraints defined as: This example is valid syntax, but we were not able to check execution
>>> cons = ({'type': 'ineq', 'fun': lambda x:  x[0] - 2 * x[1] + 2},
...         {'type': 'ineq', 'fun': lambda x: -x[0] - 2 * x[1] + 6},
...         {'type': 'ineq', 'fun': lambda x: -x[0] + 2 * x[1] + 2})
And variables must be positive, hence the following bounds: This example is valid syntax, but we were not able to check execution
>>> bnds = ((0, None), (0, None))
The optimization problem is solved using the SLSQP method as: This example is valid syntax, but we were not able to check execution
>>> res = minimize(fun, (2, 0), method='SLSQP', bounds=bnds,
...                constraints=cons)
