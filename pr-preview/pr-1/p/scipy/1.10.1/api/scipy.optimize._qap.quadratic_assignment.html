quadratic_assignment(A, B, method='faq', options=None)
$$\min_P & \ {\ \text{trace}(A^T P B P^T)}\\ \mbox{s.t. } & {P \ \epsilon \ \mathcal{P}}\\$$

Notes

Parameters

A : 2-D array, square
B : 2-D array, square
method : str in {'faq', '2opt'} (default: 'faq')
options : dict, optional

Returns

res : OptimizeResult

Examples

The see the relationship between the returned ``col_ind`` and ``fun``, use ``col_ind`` to form the best permutation matrix found, then evaluate the objective function :math:`f(P) = trace(A^T P B P^T )`. This example is valid syntax, but we were not able to check execution
>>> perm = res['col_ind']
... P = np.eye(len(A), dtype=int)[perm]
... fun = np.trace(A.T @ P @ B @ P.T)
... print(fun)
3260
Alternatively, to avoid constructing the permutation matrix explicitly, directly permute the rows and columns of the distance matrix. This example is valid syntax, but we were not able to check execution
>>> fun = np.trace(A.T @ B[perm][:, perm])
... print(fun)
3260
Although not guaranteed in general, ``quadratic_assignment`` happens to have found the globally optimal solution. This example is valid syntax, but we were not able to check execution
>>> from itertools import permutations
... perm_opt, fun_opt = None, np.inf
... for perm in permutations([0, 1, 2, 3]):
...     perm = np.array(perm)
...     fun = np.trace(A.T @ B[perm][:, perm])
...     if fun < fun_opt:
...         fun_opt, perm_opt = fun, perm
... print(np.array_equal(perm_opt, res['col_ind']))
True
