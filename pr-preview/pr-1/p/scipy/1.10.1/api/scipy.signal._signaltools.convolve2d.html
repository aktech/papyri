Examples

Compute the gradient of an image by 2D convolution with a complex Scharr operator. (Horizontal operator is real, vertical is imaginary.) Use symmetric boundary condition to avoid creating edges at the image boundaries. This example is valid syntax, but we were not able to check execution
>>> import numpy as np
... from scipy import signal
... from scipy import datasets
... ascent = datasets.ascent()
... scharr = np.array([[ -3-3j, 0-10j,  +3 -3j],
...                    [-10+0j, 0+ 0j, +10 +0j],
...                    [ -3+3j, 0+10j,  +3 +3j]]) # Gx + j*Gy
... grad = signal.convolve2d(ascent, scharr, boundary='symm', mode='same')
This example is valid syntax, but we were not able to check execution
>>> import matplotlib.pyplot as plt
... fig, (ax_orig, ax_mag, ax_ang) = plt.subplots(3, 1, figsize=(6, 15))
... ax_orig.imshow(ascent, cmap='gray')
... ax_orig.set_title('Original')
... ax_orig.set_axis_off()
... ax_mag.imshow(np.absolute(grad), cmap='gray')
... ax_mag.set_title('Gradient magnitude')
... ax_mag.set_axis_off()
... ax_ang.imshow(np.angle(grad), cmap='hsv') # hsv is cyclic, like angles
... ax_ang.set_title('Gradient orientation')
... ax_ang.set_axis_off()
... fig.show()
