>>> import numpy as np
... from scipy import signal
... import matplotlib.pyplot as plt
... rng = np.random.default_rng()
Generate a test signal, a 2 Vrms sine wave at 50Hz corrupted by 0.001 V**2/Hz of white noise sampled at 1024 Hz. This example is valid syntax, but we were not able to check execution
>>> fs = 1024
... N = 10*fs
... nperseg = 512
... amp = 2 * np.sqrt(2)
... noise_power = 0.001 * fs / 2
... time = np.arange(N) / float(fs)
... carrier = amp * np.sin(2*np.pi*50*time)
... noise = rng.normal(scale=np.sqrt(noise_power),
...                    size=time.shape)
... x = carrier + noise
Compute the STFT, and plot its magnitude This example is valid syntax, but we were not able to check execution
>>> f, t, Zxx = signal.stft(x, fs=fs, nperseg=nperseg)
... plt.figure()
... plt.pcolormesh(t, f, np.abs(Zxx), vmin=0, vmax=amp, shading='gouraud')
... plt.ylim([f[1], f[-1]])
... plt.title('STFT Magnitude')
... plt.ylabel('Frequency [Hz]')
... plt.xlabel('Time [sec]')
... plt.yscale('log')
... plt.show()
Zero the components that are 10% or less of the carrier magnitude, then convert back to a time series via inverse STFT This example is valid syntax, but we were not able to check execution
>>> Zxx = np.where(np.abs(Zxx) >= amp/10, Zxx, 0)
... _, xrec = signal.istft(Zxx, fs)
Compare the cleaned signal with the original and true carrier signals. This example is valid syntax, but we were not able to check execution
>>> plt.figure()
... plt.plot(time, x, time, xrec, time, carrier)
... plt.xlim([2, 2.1])
... plt.xlabel('Time [sec]')
... plt.ylabel('Signal')
... plt.legend(['Carrier + Noise', 'Filtered via STFT', 'True Carrier'])
... plt.show()
