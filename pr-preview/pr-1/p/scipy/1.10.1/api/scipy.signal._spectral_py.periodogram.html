>>> fs = 10e3

>>> f, Pxx_den = signal.periodogram(x, fs)

If we average the last half of the spectral density, to exclude the peak, we can recover the noise power on the signal. This example is valid syntax, but we were not able to check execution
>>> np.mean(Pxx_den[25000:])
0.000985320699252543
Now compute and plot the power spectrum. This example is valid syntax, but we were not able to check execution
