Compute the waveform with instantaneous frequency:: f(t) = 0.025*t**3 - 0.36*t**2 + 1.25*t + 2 over the interval 0 <= t <= 10. This example is valid syntax, but we were not able to check execution
>>> import numpy as np
... from scipy.signal import sweep_poly
... p = np.poly1d([0.025, -0.36, 1.25, 2.0])
... t = np.linspace(0, 10, 5001)
... w = sweep_poly(t, p)
