kaiser(M, beta, sym=True)

Notes

$$w(n) = I_0\left( \beta \sqrt{1-\frac{4n^2}{(M-1)^2}}\right)/I_0(\beta)$$
$$\quad -\frac{M-1}{2} \leq n \leq \frac{M-1}{2},$$

Parameters

M : int
beta : float
sym : bool, optional

Returns

w : ndarray
