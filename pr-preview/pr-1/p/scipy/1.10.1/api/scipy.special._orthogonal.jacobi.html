jacobi(n, alpha, beta, monic=False)
$$(1 - x^2)\frac{d^2}{dx^2}P_n^{(\alpha, \beta)} + (\beta - \alpha - (\alpha + \beta + 2)x) \frac{d}{dx}P_n^{(\alpha, \beta)} + n(n + \alpha + \beta + 1)P_n^{(\alpha, \beta)} = 0$$

Notes

Parameters

n : int
alpha : float
beta : float
monic : bool, optional

Returns

P : orthopoly1d

Examples

The Jacobi polynomials satisfy the recurrence relation: .. math:: P_n^{(\alpha, \beta-1)}(x) - P_n^{(\alpha-1, \beta)}(x) = P_{n-1}^{(\alpha, \beta)}(x) This can be verified, for example, for :math:`\alpha = \beta = 2` and :math:`n = 1` over the interval :math:`[-1, 1]`: This example is valid syntax, but we were not able to check execution
>>> import numpy as np
... from scipy.special import jacobi
... x = np.arange(-1.0, 1.0, 0.01)
... np.allclose(jacobi(0, 2, 2)(x),
...             jacobi(1, 2, 1)(x) - jacobi(1, 1, 2)(x))
True
