Notes

$$p(x; N, n, M) = \binom{n}{x} \binom{M - n}{N-x} \int_0^1 \left(1-t^{\omega/D}\right)^x\left(1-t^{1/D}\right)^{N-x} dt$$
$$D = \omega(n - x) + ((M - n)-(N-x)),$$
$$\binom{n}{k} \equiv \frac{n!}{k! (n - k)!}.$$
