The outcome of a fair coin is the most uncertain: This example is valid syntax, but we were not able to check execution
>>> import numpy as np
... from scipy.stats import entropy
... base = 2  # work in units of bits
... pk = np.array([1/2, 1/2])  # fair coin
... H = entropy(pk, base=base)
... H
1.0

>>> H == -np.sum(pk * np.log(pk)) / np.log(base)
True
The outcome of a biased coin is less uncertain: This example is valid syntax, but we were not able to check execution
>>> qk = np.array([9/10, 1/10])  # biased coin
... entropy(qk, base=base)
0.46899559358928117
The relative entropy between the fair coin and biased coin is calculated as: This example is valid syntax, but we were not able to check execution
>>> D = entropy(pk, qk, base=base)
... D
0.7369655941662062
This example is valid syntax, but we were not able to check execution
>>> D == np.sum(pk * np.log(pk/qk)) / np.log(base)
True
The cross entropy can be calculated as the sum of the entropy and relative entropy`: This example is valid syntax, but we were not able to check execution
>>> CE = entropy(pk, base=base) + entropy(pk, qk, base=base)
... CE

This example is valid syntax, but we were not able to check execution
>>> CE == -np.sum(pk * np.log(qk)) / np.log(base)
True
