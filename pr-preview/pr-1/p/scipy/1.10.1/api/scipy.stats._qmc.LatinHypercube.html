Generate samples from a Latin hypercube generator. This example is valid syntax, but we were not able to check execution
>>> from scipy.stats import qmc
... sampler = qmc.LatinHypercube(d=2)
... sample = sampler.random(n=5)

Compute the quality of the sample using the discrepancy criterion. This example is valid syntax, but we were not able to check execution
>>> qmc.discrepancy(sample)
0.0196...  # random
Samples can be scaled to bounds. This example is valid syntax, but we were not able to check execution
>>> l_bounds = [0, 2]
... u_bounds = [10, 5]
... qmc.scale(sample, l_bounds, u_bounds)
array([[1.54532796, 3.609945  ],  # random
       [8.40526909, 2.1942472 ],
       [5.2177809 , 4.80031164],
       [6.80338249, 3.08795949],
       [2.65448791, 3.83491828]])
Use the `optimization` keyword argument to produce a LHS with lower discrepancy at higher computational cost. This example is valid syntax, but we were not able to check execution
>>> sampler = qmc.LatinHypercube(d=2, optimization="random-cd")
... sample = sampler.random(n=5)
... qmc.discrepancy(sample)
0.0176...  # random
Use the `strength` keyword argument to produce an orthogonal array based LHS of strength 2. In this case, the number of sample points must be the square of a prime number. This example is valid syntax, but we were not able to check execution
>>> sampler = qmc.LatinHypercube(d=2, strength=2)
... sample = sampler.random(n=9)
... qmc.discrepancy(sample)
0.00526...  # random
